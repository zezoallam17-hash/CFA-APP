CFA Study Companion

What is included:
- index.html
- manifest.webmanifest
- sw.js
- icons/

What it does:
- Mobile-friendly CFA Level I dashboard
- Weekly plan with editable notes and completion tracking
- Visual notes with image uploads and highlighted key points
- Glossary
- Error log
- Mock tracker
- Dark mode
- Auto-save in the browser
- Export / import JSON backup

Important:
- Images are stored in the browser, so a lot of large screenshots can make storage heavy.
- Best for screenshots, diagrams, highlights, and a reasonable number of note images.

How to use it locally:
1. Unzip the folder.
2. Open the folder in any simple static server.
3. Open index.html through that local server in a browser.

Simple local server options:
- Python: python3 -m http.server 8000
- Then open http://localhost:8000

How to put it on your iPhone:
1. Upload this folder to any static host.
2. Open the hosted link in Safari on iPhone.
3. Use Add to Home Screen.
4. It will open like a small app.

Good static hosting options:
- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages

Notes:
- This package is static, so it does not need a database.
- Your data stays in the browser unless you export and import it.
